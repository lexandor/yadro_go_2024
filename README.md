# РАЗРАБОТКА МИКРОСЕРВИСНЫХ ПРИЛОЖЕНИЙ НА GOLANG

Данный репозиторий создан для сопровождения и демонстрации процесса обучения и выполнения практических заданий курса "Разработка микросервисных приложений на GoLang".

### === Сборка ===
```
make
```
или
```
make build
```

### === Запуск ===
Загрузить первые 100 комиксов (кол-во регулируется в настройках)
```
./xkcd
```

Вывести первые n записей из database.json, без флага n будет выведено первые 10 записей
```
./xkcd -o -n {кол-во записей}
```

Распечатать конфигурацию
```
./xkcd -c
```
